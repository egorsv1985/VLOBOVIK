Уникальная для Беларуси услуга

Установка с выездом

Данный вид установки автостекла ничем не уступает вклейке в нашем сервисе, так как используются только профессиональные материалы, сохраняются все гарантии.
Для установки автостекла с выездом должны быть соблюдены следующие условия:
при замене на улице
  • температура выше нуля
  • не должно быть осадков
  • необходимо 220В для зачистки пятака зеркала заднего вида
при замене в гараже
  • температура выше нуля
  • необходимо 220В для зачистки пятака зеркала заднего вида
